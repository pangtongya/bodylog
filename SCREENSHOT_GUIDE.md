## 🎨 制作要点

### 设计风格

1. **保持界面一致性**
   - 使用统一的配色方案
   - 保持一致的字体大小
   - 统一的间距和边距

2. **突出数据可视化元素**
   - 趋势图要清晰可见
   - 统计数据醒目
   - 图标和按钮突出

3. **高分辨率**
   - 所有截图必须是高清 PNG
   - 确保文字清晰可读
   - 无模糊或锯齿

4. **透明度**
   - 背景透明度 100%
   - 无水印
   - 无测试数据

### 技术要求

1. **文件格式**: PNG
2. **文件大小**: 每张不超过 5MB
3. **分辨率**: 必须符合 Apple 规格要求
4. **横屏/竖屏**: 根据设备类型
   - iPhone: 竖屏 (90度)
   - iPad: 竖屏 (90度)

### 排版建议

#### 主屏幕
```
顶部: 统计卡片
中部: 快速添加按钮
底部: 最新记录列表
```

#### 详情页

## 📸 制作方法

### 方法 1: 真机录制（推荐）
**优点**: 效果最真实，所见即所得
**步骤**:
1. 在真机上打开 App
2. 按电源键 + 音量加键（或音量减键）
3. 拍摄完整界面
4. 重复 3 次（不同界面）

### 方法 2: 模拟器截图
**优点**: 可控性强，方便调整
**步骤**:
1. 在 Xcode 中打开模拟器
2. 选择目标设备
3. 在模拟器中展示不同界面
4. Command + S 保存截图
5. 重复 3 次

### 方法 3: 委托制作
**优点**: 专业美观
**步骤**:
1. 准备设计稿（PSD、Figma）
2. 委托设计师制作
3. 确保符合 Apple 规格要求

## 🔧 常见问题

### Q1: 截图分辨率不符合要求怎么办？
A: 使用 Photoshop、Figma 或在线工具调整分辨率。

### Q2: 如何快速制作多设备截图？
A: 在 Xcode 模拟器中，按 Command + Shift + 2 快速截图。

### Q3: 截图必须包含 App Store 按钮吗？
A: 不需要。只需展示 App 内的实际界面。

### Q4: 可以用截图工具自动生成吗？
A: 可以使用 iOS 16 的"选定部分"截图工具，但要手动展示不同界面。

### Q5: 截图中的数据需要真实吗？
A: 不需要，但应合理（如体重 65-75kg 范围），避免极端数值。
