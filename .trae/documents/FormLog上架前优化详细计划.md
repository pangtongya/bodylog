## 二、核心优化方向（按优先级）

### 🔴 P0 - 必须做（上架前）

#### 1. 趋势图增加目标线（预计2小时）
**文件**: `Views/TrendView.swift`

**当前问题**: 用户不知道离目标多远

**实现方案**:
- 在Chart中添加目标线（RuleMark）
- 显示"距目标还差X"的信息
- 目标达成时显示庆祝动画

**代码改动**:
```swift
// chartCard 中添加
RuleMark(y: .value("目标", goalTargetValue))
    .foregroundStyle(Color.formlogPrimary.opacity(0.5))
    .lineStyle(StrokeStyle(lineWidth: 2, dash: [5, 5]))
    .annotation(position: .top, alignment: .leading) {
        Text("目标: \(goalDisplay)")
            .font(.system(size: 10))
            .foregroundColor(.formlogPrimary)
    }
```
